Insta Mini is a simplified Instagram-like social media application built using
the MERN stack (MongoDB, Express, React, Node.js).

## Features
- User authentication (JWT based)
- Signup & Login
- Create post with image and caption
- Feed system
- Modular backend architecture
- React Router & Context API

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js, Express
- Database: MongoDB
- Auth: JWT, bcrypt
