# Insta Mini ‚Äì MERN Project

Insta Mini is a simplified Instagram-like social media application built using the MERN stack.  
This project was developed as part of a technical assessment to demonstrate full-stack development skills.

---

## üöÄ Features

- User Signup & Login (JWT Authentication)
- Password hashing using bcrypt
- Create post with image and caption
- Feed system to view posts
- Like and Comment functionality
- Follow / Unfollow users
- Protected routes (backend & frontend)
- Clean and modular folder structure

---

## üõ† Tech Stack

### Frontend
- React (Vite) + Javascript
- React Router DOM
- Axios
- Context API
- CSS (Instagram-inspired UI)

### Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
- bcrypt
- Cloudinary (for image upload)
